Network Properties

Here you can change the network properties for this Apogee Alta Network Camera. Contact your network administrator for the appropriate settings for your network.

You must reboot in order for any network property changes to take effect.

Ethernet Information

Ethernet MAC
The Ethernet MAC address is globally unique. It also happens to be the serial number of this Apogee Alta Network Camera. This value is read-only.

DHCP Options

DHCP client

Enable this Apogee Alta Network Camera to dynamically obtain an Internet Protocol (IP) address from a Dynamic Host Configuration Protocol (DHCP) server. The DHCP server will automatically configure the camera with settings appropriate to your network.

If the DHCP client is enabled, the Static IP Options below are ignored.

Static IP Options

IP address
Enter an Internet Protocol (IP) address obtained from your network administrator or Internet service provider. An IP address is a 32-bit number that is notated by using four numbers from 0 through 255, separated by periods.

The DHCP client must be disabled for this setting to take effect.
Subnet mask
Enter the subnet mask number obtained from your network administrator or Internet service provider. This number is combined with the IP address number to identify which network segment this Apogee Alta Network Camera is on.

A subnet mask is a 32-bit number that is notated by using four numbers from 0 through 255, separated by periods. Typically, default subnet mask numbers use 0 or 255 as values (such as 255.255.255.0), but other number values can appear.

The DHCP client must be disabled for this setting to take effect.
Default gateway
Enter the IP address for the default gateway. This is the address of a local IP router on the same network as this Apogee Alta Network Camera that is used to forward traffic to destinations beyond the local network. Use four numbers from 0 through 255, separated by periods.

A gateway is a router that connects separate IP network segments. For example, a network segment might need a gateway to connect it to another network segment, a wide area network (WAN), or to the Internet.

The DHCP client must be disabled for this setting to take effect.

Apogee Alta Network Camera Options

HTTP server port
Enter the Apogee Alta Network Camera server port number. Default = 80 (TCP/HTTP).

Please note that web browsers use port 80 for HTTP traffic. Changing the port number requires the new port to be explicitly specified when when entering a URL into the web browser. Apogee Instruments does not recommend changing this setting unless you have a specific reason for doing so. If a port change is desired, the port 2571 is suggested as an alternate.
Camera ID
Enter the Apogee Alta Network Camera ID string. Default = "Apogee1".

This is a 31 character descriptive identifier. It is only used for the discovery of cameras on the LAN. It does not have to be set to this camera's host name.
 

Serial Port Properties

Here you can change the properties of both serial port A and B for this Apogee Alta Network Camera. First select the serial port you wish to configure, then change the settings for the selected serial port.

You must reboot in order for any serial port property changes to take effect.

Serial port
Select the Apogee Alta Network Camera serial port to configure. You must press the Select button for the selection to take effect.

Serial Port IP Options

Serial IP protocol
Select the network connection type. Default = Disabled.
Serial IP port
Enter the network service port number. Default A = 2572 and B = 2573.
Serial IP maximum packet size
Enter the maximum number of bytes to be received from the serial port before transmitting it across the network. Default = 512 bytes.
Serial IP connection timeout
Enter the number of seconds to wait for a character to be received from the network before the TCP network connection is closed. This timeout is approximate and will be more inaccurate the larger the timeout value is. Default = 86400 seconds (24 hours).

Serial Port Options

Serial bit rate
Select the bit rate in bits per second. Default = 9600 bps.
Serial data bits
Select the number of data bits. Default = 8.
Serial parity bits
Select the parity bit information. Default = None.
Serial stop bits
Select the number of stop bits. Default = 1.
Serial flow control
Select the type of flow control. Default = None.
 

Change Login Password

Here you can change the login password for this Apogee Alta Network Camera.

You must reboot in order for this change to take effect.

login password
Each user ID requires a case-sensitive password in order to configure the Apogee Alta Network Camera. Refer to the Apogee Alta Network Camera manual for the initial user ID and password.
verify password
Enter the login password again to verify that there are no errors.
Session timeout
Enter the number of seconds to wait for a request before the login session times out. You may need to close and restart your browser to log back in if the session timeout is reached. Default = 420 seconds.

Many configuration changes only take effect when the Apogee Alta Network Camera is rebooted.

Reboot
Reboot the Apogee Alta Network Camera. Many configuration changes only take effect when the Apogee Alta Network Camera is rebooted.
